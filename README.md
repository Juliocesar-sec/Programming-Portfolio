# Programming Learning Path

Programming is the thread that connects industries together. Whether you want to build tools for healthcare, design financial applications, develop enterprise software, or create the next successful game, programming provides the foundation to transform ideas into reality.

This learning path is designed to build strong programming fundamentals through multiple industry-relevant languages. Each course focuses on practical skills, problem-solving techniques, software development principles, and hands-on projects that reinforce real-world programming concepts.

## Course Overview

### 1. Python

**File:** `Python`

Learn the fundamentals of computer programming using Python, one of the world's most versatile and beginner-friendly programming languages.

#### Topics Covered

* Introduction to programming concepts
* Variables and data types
* Operators and expressions
* Conditional statements
* Loops and iteration
* Functions and modular programming
* Error handling basics
* Problem-solving techniques
* Programming best practices

#### Skills Gained

* Design and develop basic applications
* Analyze and solve programming problems
* Write clean and maintainable code
* Understand core software development principles

#### Final Project

**Personal Expense Tracker**

Build a console-based application that allows users to:

* Add income and expenses
* Categorize transactions
* View spending summaries
* Calculate balances
* Generate simple financial reports

---

### 2. JavaScript

**File:** `JavaScript`

Learn the foundations of JavaScript and discover how modern interactive web applications are built.

#### Topics Covered

* JavaScript fundamentals
* Variables and data types
* Operators and expressions
* Conditional logic
* Loops and iteration
* Functions
* Arrays and objects
* Basic debugging
* Programming best practices

#### Skills Gained

* Create interactive web functionality
* Build dynamic applications
* Understand client-side programming
* Develop problem-solving abilities

#### Final Project

**Interactive To-Do Application**

Build a web-based task manager that allows users to:

* Add tasks
* Edit tasks
* Delete tasks
* Mark tasks as completed
* Filter completed and pending items

---

### 3. C++

**File:** `C++`

Develop strong programming fundamentals while learning C++, a language widely used in finance, gaming, networking, and high-performance systems.

#### Topics Covered

* Compilation process
* Variables and data types
* Operators
* Control flow
* Loops
* Functions
* Pointers
* Memory management fundamentals
* Data conversions

#### Skills Gained

* Understand low-level programming concepts
* Manage memory effectively
* Build efficient applications
* Strengthen problem-solving and analytical skills

#### Final Project

**Student Management System**

Build a command-line application that can:

* Register students
* Store grades
* Calculate averages
* Search student records
* Generate performance reports

---

### 4. C

**File:** `C`

Explore one of the most influential programming languages and gain a deeper understanding of how software interacts with computer systems.

#### Topics Covered

* C language syntax
* Variables and data types
* Operators and expressions
* Conditional statements
* Loops
* Functions
* Arrays
* Pointers
* Memory concepts
* Program organization

#### Skills Gained

* Develop efficient programs
* Understand system-level programming concepts
* Work with memory directly
* Build strong software engineering foundations

#### Final Project

**Library Management System**

Build a console-based system that allows users to:

* Add books
* Search books
* Borrow books
* Return books
* Track inventory records
